  Lightbox problem
----------------------------------------------------------------------------

  This theme is using jQuery.So if you wan't to use lightbox try WP-Slimbox plugin.
 
  WP-Slimbox2 (lightbox based on jQuery)
  http://wordpress.org/extend/plugins/wp-slimbox2/


  add-lightbox plugin will add rel="lightbox" to your image automatically.

  add-lightbox
  http://wordpress.org/extend/plugins/add-lightbox/
